# 90 Days DevOps Learning Plan

## About Me
I am currently a working professional with experience in Salesforce and deployment-related activities. I want to learn DevOps and Cloud Engineering by building strong hands-on skills in Linux, automation, CI/CD, containers, and cloud platforms.

---

## My Understanding of DevOps & Cloud Engineering
DevOps is a culture and engineering practice that focuses on automation, collaboration, continuous integration, continuous delivery, monitoring, and faster software delivery with reliability.

Cloud Engineering is about building, deploying, scaling, and managing applications and infrastructure using cloud platforms like AWS, Azure, or GCP.

DevOps and Cloud together help organizations release software faster, reduce manual effort, improve stability, and scale efficiently.

---

## Why I Am Learning DevOps
- To move toward a high-growth engineering career
- To gain hands-on infrastructure and automation skills
- To work on real-world deployment, CI/CD, Kubernetes, and cloud projects
- To become confident in production troubleshooting and deployment automation

---

## My 90-Day Goals
1. Build strong Linux, Git, and scripting fundamentals
2. Create and deploy CI/CD pipelines using Jenkins and GitHub Actions
3. Deploy a production-style application using Docker, Kubernetes, and AWS
